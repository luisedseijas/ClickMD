# Ejemplos de Moléculas

Coloca moléculas de ejemplo aquí para desarrollo y testing.

**Fuentes recomendadas:**
- [PubChem](https://pubchem.ncbi.nlm.nih.gov/)
- [Protein Data Bank](https://www.rcsb.org/)
- [ChemSpider](http://www.chemspider.com/)

**Moléculas sugeridas para testing:**
1. **Metano** (CH4) - Molécula más simple
2. **Etano** (C2H6) - Pequeña, alkano
3. **Benceno** (C6H6) - Aromático simple
4. **Glicerol** (C3H8O3) - Molécula con oxígeno
5. **Alanina** (C3H7NO2) - Aminoácido simple
